# Stage Starz Academy of Dance Website

Mobile-responsive static website prepared for GitHub Pages.

## Publish with GitHub Pages

1. Create a new public GitHub repository named `stage-starz-website`.
2. Upload all files from this folder to the repository root.
3. Open **Settings > Pages** in the repository.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then click **Save**.
6. GitHub will provide a public URL such as:
   `https://YOUR-USERNAME.github.io/stage-starz-website/`
