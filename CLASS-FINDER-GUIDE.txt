STAGE STARZ CLASS FINDER — CORRECTED ROUTING

The recommendation logic has been rewritten so each age has an explicit route:

- Ages 3–4 -> Preschool Program
- Ages 5–6 -> Primary Program
- Ages 7–9 -> Elementary Program
- Ages 10–12 -> Intermediate & Advanced Program
- Age 13 or older -> Intermediate & Advanced Program (teen pathway)

Intentional overrides:
- Competition goal, interest, or prior competition experience -> Competition Team Auditions
- Specialty-style interest -> Specialized Classes

The result now displays the visitor's selected answers so families and staff can verify that the correct age and preferences were read.

There is no longer any preschool fallback. If an age value is missing or invalid, the tool routes to staff-assisted placement rather than recommending Preschool.
